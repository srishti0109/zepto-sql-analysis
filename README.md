# 🛒 Zepto SQL Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing Zepto product data using SQL.  
The dataset contains product details such as product name, category, price, discount, stock availability, and ratings.

The goal of this project is to perform:
- Data cleaning
- Data transformation
- Business insight analysis using SQL queries

---

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- Kaggle Dataset
- GitHub

---

## 📚 SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- DISTINCT
- UPDATE
- ALTER TABLE
- Aggregate Functions
- Data Cleaning

---

## 🧹 Data Cleaning Performed
- Checked null values
- Added primary key column
- Converted product prices from paise to rupees
- Changed column data types for decimal support

---

## 📊 Business Questions Solved
- Finding products with highest MRP
- Checking stock availability
- Identifying high discount products
- Product price analysis
- Dataset exploration using SQL queries
- Revenue analysis
